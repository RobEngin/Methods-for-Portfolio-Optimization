EuroStoxx50DS FOLDER

The folder contains the weekly prices data adjusted for dividends for the Eurostoxx50 from 07-12-2005 to 06-24-2014.
These prices are available in .xlsx extension.

The sheet 'Assets' contain a matrix where each column is the weekly prices of each asset.

The sheet 'Index' contain a column vector which is the weekly prices of the market index.
